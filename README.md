🎯 QuizForge: C++ Based Interactive Quiz System

📌 Overview
QuizForge is a terminal-based interactive quiz application developed in C++ that provides a structured and engaging platform for users to test their knowledge through multiple-choice questions. The project is designed to go beyond basic quiz programs by incorporating persistent user tracking, ensuring that scores and progress are maintained across multiple sessions. The application uses object-oriented programming principles to create a modular and organized system, with separate classes handling users, questions, quiz logic, and scoring. It features a user authentication system where players can register with a unique username or log in to continue their progress. All user data, including high scores, is stored using file handling, allowing the system to update records automatically when a user achieves a better score. QuizForge offers multiple gameplay modes, including a timed mode for added challenge and a non-timed mode for relaxed play. Users can also select from different difficulty levels—easy, medium, and hard—making the application suitable for a wide range of skill levels. A dynamic scoring system rewards correct answers and penalizes incorrect ones, encouraging accuracy and competitiveness. The program follows a menu-driven interface, ensuring smooth navigation between features such as login, mode selection, difficulty selection, and replay options. Internally, it utilizes core programming concepts like vectors for dynamic data storage, loops for question handling, and conditional logic for decision-making. Overall, QuizForge demonstrates practical implementation of key C++ concepts such as object-oriented design, file handling, and structured program flow, making it both an educational project and a functional quiz system.

🚀 Features
User Registration & Login
Easy, Medium, Hard difficulty levels
Timed and Non-Timed modes
Dynamic scoring system (+4 / -1)
High score tracking
Menu-driven interface
Data saved using files

⚙️ Working
User logs in or registers
Selects game mode
Chooses difficulty level
Answers quiz questions
Score is calculated
High score is updated in file
User can replay or exit
